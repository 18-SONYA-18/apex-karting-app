# 🏎️ Апекс Картинг — Веб-приложение

## О проекте

**Апекс Картинг** — клиентское веб-приложение для самостоятельной записи на заезды уличного картинг-центра. Позволяет клиентам просматривать расписание, бронировать места, выбирать экипировку и управлять своими записями.

Проект адаптирован из исходной версии «Волна» (SUP-прогулки) под специфику картинг-центра.

---

**Frontend:**

## ✨ Основные функции

### 🔐 Авторизация
- Вход по номеру телефона (OTP через SMS)
- Регистрация новых клиентов
- Управление сессией (access/refresh токены)

###  Заезды
- Просмотр списка доступных заездов
- Фильтрация по дате, типу трассы, маршалу
- 
### 🎫 Бронирование (не удалось)
- Выбор числа мест (до 4 человек)
- Выбор экипировки (своя/прокатная)
- Автоматический расчёт стоимости
- Защита от двойного бронирования
- Идемпотентность запросов

###  Мои записи
- Просмотр предстоящих и прошедших заездов (+)
- Детали брони (трасса, маршал, экипировка, цена) (-)
- Отмена по правилу 2 часов (-)
- Статусы: активна, отменена, поздняя отмена, отменена центром (-)

### 👤 Профиль (-)
- Просмотр и редактирование имени
- Смена телефона (с подтверждением OTP)
- Выход из аккаунта
- Удаление аккаунта

### 🗺️ Дополнительно (что хотелось бы, но не вышло)
- Карта центра с переходом в Яндекс.Карты
- Push-уведомления (запрос разрешения)
- Добавление заезда в Google Calendar

## 🚀 Запуск

### 1. Установка зависимостей

```bash
cd 02-development/backend
pip install -r requirements.txt
```

### 2. Запуск сервера

```bash
python app.py
```

Сервер запустится на `http://localhost:5000`

### 3. Открытие приложения

Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## 🔑 Тестовые данные

При первом запуске автоматически создаются:
- **Маршалы:** Алексей, Дмитрий, Ольга
- **Трассы:** Короткая (новичковая), Длинная (опытная)
- **Заезды:** 2 тестовых заезда на ближайшие дни

### OTP код

Код подтверждения выводится в консоль сервера:
```
SMS OTP for +79991234567: 1234
```

##  Бизнес-правила

### Лимиты
- Максимум мест в брони: **4** (себя + до 3 гостей)
- Лимит трассы: зависит от конфигурации (8-14 мест)
- Прокатная экипировка: ограничена наличием

### Отмена
- **Ранняя отмена** (≥2 часов до старта): место освобождается
- **Поздняя отмена** (<2 часов): место не освобождается, штрафов нет
- **После старта:** отмена недоступна

### Цены
- Оплата на месте (наличные/перевод)
- Формула: `цена × места + прокат × прокатные`
